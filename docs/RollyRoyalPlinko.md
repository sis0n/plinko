# RollyRoyal Plinko – 2d arcade theme game

## Description
RollyRoyal Plinko is a physics-based digital Plinko game developed using the HTML5 Canvas API. The system simulates a ball dropping through a grid of pegs, where its trajectory is dynamically calculated using real-time gravity and collision physics. It serves as a technical showcase of 2D coordinate transformations, featuring a luxury casino aesthetic built with procedural gradients and interactive animations. A key feature is the "Digital Receipt" system, which uses toDataURL to allow users to export their winning moments as image files.

## Objectives
*   **Technical Implementation:** Execute advanced Canvas API functions including translate, rotate, scale, and transform for all game assets.
*   **Physics Simulation:** Develop a custom engine to handle realistic gravity, velocity, and bounce (restitution) during ball-and-peg collisions.
*   **Visual Design:** Create a high-end casino atmosphere using radial and linear gradients for the game board and UI.
*   **Data Portability:** Enable the toDataURL method to capture and download the current game state as a PNG file.
*   **Animation Quality:** Achieve a smooth 60FPS rendering loop using requestAnimationFrame.

## Scope
*   **Dynamic Physics:** Real-time calculation of ball movement and peg collisions.
*   **Interactive Controls:** User-selected drop points for strategic gameplay.
*   **UI Animations:** Animated "Jackpot" effects using scaling and rotation triggers.
*   **Export Feature:** Functional "Save Result" button to convert the canvas into a downloadable image.

## Limitations
*   **No Real Currency:** Designed strictly for educational and simulation purposes.
*   **Single-Player:** Localized experience with no multiplayer or online leaderboards.
*   **Fixed Board Layout:** The peg arrangement is static and cannot be customized by the user.
