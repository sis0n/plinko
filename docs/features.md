# Project Features - RollyRoyal Plinko

## Core Gameplay & Physics
- **Custom Physics Engine:** Real-time simulation of gravity, velocity, and restitution (bounce) for realistic ball-peg interactions.
- **Dynamic Collision Detection:** Precise calculation of trajectories when the ball hits pegs.
- **Interactive Drop Points:** Allows users to strategically select where to drop the ball on the grid.

## Graphics & Animation
- **HTML5 Canvas Rendering:** High-performance 2D rendering for all game elements.
- **Advanced Transformations:** Utilization of `translate`, `rotate`, `scale`, and `transform` for dynamic UI elements and animations.
- **Luxury Casino Aesthetic:** Use of procedural radial and linear gradients to create a premium visual experience.
- **60FPS Animation Loop:** Smooth rendering powered by `requestAnimationFrame`.
- **Jackpot Effects:** Special UI animations triggered during winning moments.

## Functionality & Export
- **Digital Receipt System:** Ability to export the current game state or winning moment as a PNG image using the `toDataURL` method.
- **Responsive Layout:** Centered game board using canvas translations.

## Technical Requirements Fulfillment
- **Comprehensive Canvas API Usage:** Implementation of all required transformation methods (`scale`, `transform`, `translate`, `rotate`).
- **Data Portability:** Integrated image export functionality.
